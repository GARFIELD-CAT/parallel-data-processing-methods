# Практическое задание 1 — Основы многопоточности в .NET

Кратко:

- Реализованы последовательная и многопоточная обработки массива decimal.
- Многопоточная версия использует низкоуровневый Thread API и ManualResetEvent для синхронизации.
- Измерение времени происходит через Stopwatch.

Файлы:

- Program.cs — точка входа, генерирует 10_000_000 элементов (seed=42), запускает обе версии и выводит статистику.
- DataProcessor.cs — методы ProcessDataSequential и ProcessDataParallel.
- PerformanceMeter.cs — MeasureExecutionTime и CompareResults.
- README.md — этот файл.

Результаты (примерные, зависят от машины):

- Последовательная обработка: ~X мс
- Параллельная обработка (4 потока): ~Y мс
- Ускорение: ~ (X/Y)x
- Результаты совпадают: Да (при tolerance=0.0001)

Тестирование:

- Перед финальным запуском рекомендую запускать с меньшими размерами (1000, 10000, 100000) для валидации корректности.

Замечания по реализации:

- Данные не модифицируются совместно; каждый поток пишет в отдельные диапазоны результата.
- Корректное ожидание потоков реализовано через ManualResetEvent и WaitHandle.WaitAll.
- Исключения внутри потоков собираются и пробрасываются в основной поток как AggregateException.
